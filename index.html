Write down difference between Declaration and Initialization a variable.

Creating a variable in JavaScript is called "declaring" a variable.Declaration basically means introducing a new entity to the program.To declare (create) a variable, we need to use the var, let, or const keyword, followed by the name we want to provide to the variable. The var, let, and const keywords tell JavaScript to set aside a portion of memory so that we may store a specific piece of data to it later.
like Declare a variable named "x" using the var keyword
var x;
Initialization is giving a variable it's first value. Initialization is the process of assigning a value to the Variable. Storing a value in a variable is called variable initialization
like Assign a value of 2 to the variable "x" using the assignment operator (=)
x=2;

Task # 1: In the below code, we are using only one CSS property at a time, so my question is, Can we add multiple CSS properties when we are using getElementById?

Code : document.getElementById("jspara").style.backgroundColor = "yellow"; document.getElementById("jspara").style.fontSize = "20px"; document.getElementById("jspara").style.color = "red";

Yes! It is very much easy to set multiple styles for an element in a single statement using setAttribute property. It doesn't effect the existing properties and avoids the complexity of going for loops or plugins.



document.getElementById("jspara").setAttribute("style", " background-color: yellow; font-size: 20px; color: red;" )
